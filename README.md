# Learning-platform
Inclusive Learning Platform is a web-based learning system developed using WordPress and PHP that aims to provide an accessible and adaptive educational environment for all students, especially focusing on differently-abled users. The platform offers features like adaptive learning paths that suggest course materials based on student progress, interactive quizzes, video tutorials with captions, and downloadable resources. It includes important accessibility options such as screen reader support, keyboard navigation, and high-contrast mode to ensure ease of use for everyone. The system also supports multiple regional languages, allowing users to learn in their preferred language. Built on WordPress for easy content management and powered by custom PHP for adaptive logic and progress tracking, this platform promotes inclusive education and helps bridge learning gaps in a user-friendly manner.
